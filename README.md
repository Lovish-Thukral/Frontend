# 🎨 NextepAI Frontend

Frontend application for NextepAI — a modern AI-powered platform delivering an interactive and responsive user experience.

---

## 📌 Overview

The NextepAI Frontend provides:
- Clean and responsive UI
- Seamless interaction with backend APIs
- Real-time data rendering
- User-friendly design for AI-based workflows

---

## ⚙️ Tech Stack

- **Framework:** React / Next.js
- **Language:** JavaScript / TypeScript
- **Styling:** Tailwind CSS / CSS Modules
- **State Management:** Redux / Context API
- **API Handling:** Axios / Fetch
- **Build Tool:** Vite / Webpack

---

## 📂 Project Structure

```
Frontend/
│── public/             # Static assets
│── src/
│   │── components/     # Reusable UI components
│   │── pages/          # Application pages/routes
│   │── services/       # API calls
│   │── hooks/          # Custom hooks
│   │── context/        # Global state management
│   │── utils/          # Helper functions
│── package.json        # Dependencies
│── README.md
```

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Lovish-Thukral/Frontend.git
cd Frontend
```

---

### 2. Install Dependencies

```bash
npm install
```

---

### 3. Setup Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:5000
```

---

### 4. Run the Development Server

```bash
npm run dev
```

---

## 🔗 API Integration

The frontend communicates with the NextepAI backend via REST APIs.

Example:

```
GET /api/status
POST /api/process
```
